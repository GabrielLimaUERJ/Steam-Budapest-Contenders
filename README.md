# CS2 Budapest Sticker Analyzer

Aplicação interativa em Python para análise de preços de stickers dos times Contenders do Major Budapest 2025 de CS2 no Steam Market, permitindo comparar valores entre diferentes itens.

## Tecnologias

- Python  
- Streamlit  
- Pandas  
- Matplotlib  
- API pública do Steam Market

## Funcionalidades

- Seleção de até 16 stickers  
- Consulta automática de preços no Steam Market  
- Geração de tabela de comparação  
- Cache de requisições para evitar bloqueio da API
